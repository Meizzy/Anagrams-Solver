# Anagrams-Solver
An app that returns a request from a remote API that solves your anagram puzzle with all the possible combination of words.

## Technologies used:
* Kotlin
* Android Studio
* XML
* MVVM & CLEAN architecture
* KotlinX Coroutines
* Material Design (Google)
* SearchView
* Anagramica API
* Retrofit2
* RecyclerView (multiple viewType)
